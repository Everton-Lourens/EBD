# Como excluir um aluno

## Passos
1. Abrir a página de edição do aluno.
2. Conferir se é realmente o registro correto, porque a exclusão remove o aluno da base.
3. Usar o botão **Excluir Aluno** no topo da página, ao lado de **Voltar**.
4. Confirmar a ação no alerta de segurança.
5. Aguardar o retorno para a tela principal e atualizar a listagem local.

## Observação
O backend usa a ação `deletealuno` e remove o registro localizado pelo identificador atual do aluno.
