# Desenvolvimento local com Express

## Passos
1. Instale as dependências com `npm install`.
2. Execute `npm start` para subir o servidor local.
3. Abra `http://localhost:8080/` para a tela de login.
4. Após autenticar, acesse o dashboard em `http://localhost:8080/src/modules/dashboard/pages/home/index.html`.
5. Para testar atalhos de navegação, use rotas como `http://localhost:8080/chamada`.

## Observação
O servidor local deve servir o diretório raiz do projeto para manter compatibilidade com os caminhos estáticos usados no GitHub Pages.
