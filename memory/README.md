# Memória viva do projeto EBD

Esta pasta concentra o conhecimento consolidado do projeto. A ideia é manter as decisões, os procedimentos operacionais, as armadilhas conhecidas e as regras de negócio em arquivos pequenos, legíveis e versionáveis.

A aplicação também mantém um espelho em `src/js/services/memory.js`, que registra sessões curtas e consolida os principais aprendizados no navegador.
